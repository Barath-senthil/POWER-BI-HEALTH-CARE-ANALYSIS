# 🏥 Hospital Analytics Dashboard (Power BI)

## 📊 Project Overview

This project presents a set of **interactive Power BI dashboards** designed to analyze hospital operations, patient health insights, and doctor performance.

The dashboards provide a comprehensive view of hospital data to support better decision-making using data visualization.

---

## 🛠 Tools Used

* Power BI
* Data Modeling
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation
* Interactive Visualizations

---

## 📁 Dashboards Included

### 🔷 1. Hospital Overview Dashboard

* Total Patients, Appointments, Doctors, and Bill Value KPIs
* City-wise Gender Distribution
* Department-wise Revenue
* Appointment Details Table
* Filters: Gender, Status, City, Department

---

### 🔷 2. Patient Health Dashboard

* Total Issues, High Severity Cases, Emergency %
* Month-wise Appointment Trends
* Department-wise High Severity Analysis
* Visit Type (OP vs Emergency)
* Issue-wise Patient Data
* Filters: Status, Severity, Issue

---

### 🔷 3. Doctor Performance Dashboard

* Total Doctors, Salary, Bill Value, Appointments
* Department-wise Patient Load
* Revenue vs Salary Comparison
* Doctor Salary vs Experience Trend
* Department Revenue Contribution
* Filters: City, Department

## 🎯 Key Insights

* General and Dermatology departments generate higher revenue
* Emergency and high severity cases show consistent patterns
* Doctor experience impacts salary and performance
* City-wise distribution highlights patient concentration
